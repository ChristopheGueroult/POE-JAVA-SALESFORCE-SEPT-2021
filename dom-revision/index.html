Formation Fullstack

Dans ce Boot camp de 30 semaines, nous allons se faire de vous des véritables développeurs Fullstack au travers d'un parcours unique où vous allez non seulement développer des compétences incroyables sur l'intégration et développement des UI, mais vous allez aussi devenir des développeurs backent aguerris.

Parcours Frontend

Les technos qui seront abordés pour la première partie, à savoir le parcours Frontend sont ;

  1. Fonctionnement du réseau et d'internet
  2. Fonctionnement des sites internet
  3. Le HTML5
  4. Le CSS3
  5. Le SASS
  6. Le Javascript
  7. Typescript
  8. Algorithmes et structures de données
  9. Introduction à React.js
  10. React.js avancé
  11. L'architecture Redux

HTML5

C'est le langage qui va nous permettre de structurer une page web.

CSS 3

C'est le langage qui va nous permettre d'en mettre en place le style, c'est-à-dire le côté esthétique d'une page web.

Le SASS

C'est un langage de programmation qui va nous permettre d'écrire un code CSS modularisé avec plus de fonctionnalité, et qui va ensuite être compilé en CSS pour être compris par le navigateur.

Javascript

C'est le langage qui va donner vie et intelligence à nos pages web. Il va donc ajouter de l'interactivité pour dynamiser la page de telle sorte que le contenu soit mis à jour selon les actions de l'utilisateur.

Typescript

C'est un langage de programmation fortement typé qui va nous permettre d'écrire du code Frontend robuste pour ensuite être compilé en un Javascript propre.

Algorithmes et structures de données

Le but sera d'aiguiser la capacité à résoudre des problèmes et écrire du code optimisé en se basant sur l'optimisation dans l'utilisation des ressources.

Introduction à React.js

C'est une librarie Javascript créée par Facebook et qui permet de développer les interfaces utilisateur par les moyens de blocs de code réutilisable appelés composants qui représentent une partie ou des éléments d'interface utilisateur.

React.js avancé

Nous allons amener vos compétences React js a un autre niveau.

L'architecture Redux

Redux est une librairie Javascript qui va vous permettre de centraliser la gestion de l'État au sein de vos applications React js.

Parcours Backend

Les technologies qui seront abordés pour la deuxième partie, à savoir le parcours Backend sont :

  1. Cycle Requête/Réponse dans la communication client-serveur(web)
  2. Bases de Node.js et l'écosystème NPM
  3. Module native de Node.js : HTTP,FS,PATH,HTTPS,OS
  4. Mise en place d'un serveur web avec Node.js
  5. Mise en place d'un serveur web avec Express.js
  6. Gestion des routes avec avec Express.js
  7. Utilisation de Express.js et Typescript
  8. Mise en place des websocket avec socket.io
  9. Base de données NoSQL avec MongoDB
  10. Base de données relationnelles avec MySQL
  11. Développement d'une interface REST pas à pas avec Node.js & Express avec Utilisant MongoDB

Projet Final : MERN Stack

Le projet final est le projet par lequel l'étudiant validera l'acquisition des compétences Fullstack.Voici les exigences du projet :

  1. Equipe de 3 développeurs
  2. Durée : 2 mois
  3. React.js & Redux,Typescript pour le Frontend
  4. Node.js,Express, Typescript pour le backend
  5. MongoDB ou MySQL comme base de données : Ici, il faut justifier le choix de la BDD
  6. Redis pour implémenter la logique de caching coté backend selon le besoin
  7. Mise en place des tests automatisés avec une couverture d'au moin 90% sur le front/backend et présenter le plan de test
